Подключите console.UIAction классы к главному приложению.
Для этого на данный момент можно создать статические поля:

	private static  java2.realestate.core.database   .java2.realestate.core.database = new InMemoryDatabaseImpl();
	private static console.UIAction addUIAction = new AddXXXUIAction(  .java2.realestate.core.database);
	private static console.UIAction removeUIAction = new RemoveXXXUIAction(  .java2.realestate.core.database);
	private static console.UIAction getAllUIAction = new GetAllXXXUIAction(  .java2.realestate.core.database);
	private static console.UIAction exitUIAction = new ExitUIAction();

и воспользуйтесь этими классами в коде приложения.
Код перенесённый в console.UIAction классы нужно удалить из главного приложения.
